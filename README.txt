
Het runnen van de applicatie: 

Om het applicatie te gebruiken moet u eerst de statements.sql bestand importen via de localhost. 
Nadat u de database heeft aangemaakt worden er een paar voorbeelden toegevoegd. Dit zijn aanpasbaar. 
Daarna moet u de populate.php file een keer openen via u browser zodat er een admin user wordt aangemaakt 
met de username: admin en password: admin. Hierna kunt u inloggen via de index.php file en de password aanpassen.
